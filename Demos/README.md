# Demos

## Demo 1
Synchroner Methodenaufruf. Das Hauptprogramm führt eine Zählschleife mit 100 Durchgängen durch und nach der Hälfte der Durchgänge wird die Methode aufgerufen, die etwas ähnliches macht. Dabei wird die Abarbeitung des Hauptprogramms angehalten und dann die Methode ausgeführt. Nach Beendigung der Methode wird das Hauptprogramm fortgesetzt, wo es zum Aufruf der Methode angehaltet worden ist.
